# 🧸 我的待办清单

一个使用 HTML、CSS 和 JavaScript 编写的简洁待办事项网页工具。

## 在线预览
https://nomeain.github.io/todo-app/

## 项目简介
这是一个适合日常使用的轻量级待办事项网页应用，界面简洁，操作直观，支持本地存储，刷新页面后任务不会丢失。

## 功能特性
- 添加任务
- 使用回车键快速添加任务
- 标记任务为已完成 / 未完成
- 已完成任务自动显示删除线
- 删除单条任务
- 清空所有已完成任务
- 高 / 中 / 低优先级设置
- 不同优先级使用不同颜色标识
- 支持行内编辑任务内容
- 显示任务添加时间
- 显示未完成和已完成任务数量
- 空列表时显示友好提示
- 响应式设计，支持手机端使用
- 使用 localStorage 进行本地存储

## 技术栈
- HTML
- CSS
- JavaScript
- localStorage
